Architecture projects

Runtime / research · Rust in progress

Traverse

Traverse is where the UMA and C-DAD ideas go further. A contract-driven Rust and WASM runtime for discovering, validating, and composing portable business capabilities across multi-agent orchestration scenarios. The spec is governed, the quality standards are in place, and the ADR process is documented before a single line of implementation is written. That is not bureaucracy. That is the only way to build a runtime that stays coherent as it grows.

Personal knowledge layer · Rust, WASM, Web Components, PWA in progress

youaskm3

A personal knowledge layer built on the same foundations as Traverse and UMA. WASM-native, MCP-powered, designed to work with agentic systems. Runs entirely on GitHub Pages with no server, no database, and no ongoing cost. The federation model means anyone can fork it, run their own instance, and register in a shared registry.
